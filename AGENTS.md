# AGENTS.md — AI 에이전트 공통 작업 규칙

이 문서는 이 워크스페이스에서 작업하는 모든 AI 에이전트가 따르는 **공통 규칙**입니다.
프로젝트별 세부 지침은 `00_context/AI_INSTRUCTIONS.md`를 함께 참조하세요.

---

## 1. AI의 역할

- AI는 **최종 의사결정자가 아니라 분석/작성 보조자**다.
- 모든 AI 산출물은 **초안(draft)**이며, 사람이 최종 검토·승인하기 전까지 확정본이 아니다.
- 판단이 필요한 부분은 결정하지 말고 `06_decisions/OPEN_QUESTIONS.md`에 질문으로 남긴다.

---

## 2. 작업 시작 전 필수 확인

작업을 시작하기 전에 반드시 `00_context/`의 문서를 다음 순서로 확인한다.

1. `PROJECT_OVERVIEW.md` — 프로젝트 목적·범위
2. `AI_INSTRUCTIONS.md` — 프로젝트별 작업 지침
3. `SECURITY_RULES.md` — 보안 규칙
4. `WORKFLOW.md` — 표준 작업 흐름
5. `OUTPUT_STANDARDS.md` — 산출물 표준
6. `GLOSSARY.md` / `DOCUMENT_INDEX.md` — 용어·문서 색인

---

## 3. 폴더 사용 규칙

- 입력자료는 `01_inputs/`를 **우선 참조**한다. 임의로 외부 지식으로 대체하지 않는다.
- 요청 단위 작업은 `02_requests/`에서 관리한다(`pending` → `in_progress` → `completed`).
- 중간 분석과 초안은 `03_working/`에 둔다.
- 공유 가능한 결과물은 `04_outputs/`에 둔다.
- 피드백과 승인 기록은 `05_feedback/`에 둔다.
- 결정사항과 미해결 질문은 `06_decisions/`에 남긴다.
- 프롬프트는 `07_prompts/`에 보관한다.
- 오래된 작업물은 `08_archive/`로 옮긴다.

---

## 4. 신뢰도 표기 규칙

모든 결과에는 정보의 신뢰도를 다음 4가지로 구분해 표기한다.

| 표기 | 의미 |
|------|------|
| `확정` | 입력자료·결정 로그 등 근거로 확인된 사실 |
| `추정` | 근거로부터 합리적으로 유추했으나 확인되지 않은 내용 |
| `확인필요` | 판단이 필요하거나 사람에게 물어봐야 하는 내용 |
| `미확인` | 근거가 없거나 정보가 부족한 내용 |

- **근거 없는 내용을 `확정` 표현으로 쓰지 않는다.**
- 추정에는 근거와 가정을 함께 적는다.

---

## 5. 입력자료 취급 규칙

- 원본 입력자료(`01_inputs/`, 특히 `raw/`)를 **임의로 수정·삭제하지 않는다.**
- 가공이 필요하면 원본을 복사해 `03_working/`에서 작업한다.
- 인용 시 출처(파일명/문서ID)를 남긴다.

---

## 6. 보안 규칙 (요약)

- 개인정보·실데이터·계정/비밀번호/API Key/토큰/접속정보를 저장하지 않는다.
- 내부 IP·서버 정보를 기록하지 않는다.
- 예시 데이터는 가상 값만 사용한다.
- 상세 규칙은 `00_context/SECURITY_RULES.md`를 따른다.

---

## 7. 산출물에 항상 남길 정보

각 산출물에는 다음을 포함한다(자세한 항목은 `AI_INSTRUCTIONS.md`).

- 작업 목적
- 사용한 입력자료
- 판단 근거
- 확정 사항 / 추정 사항 / 확인 필요 사항
- 다음 작업
